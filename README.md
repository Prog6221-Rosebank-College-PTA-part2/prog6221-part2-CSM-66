CyberSecurity Awareness Bot

The CyberSecurity Awareness Bot is a Windows Forms app built in C#. It's designed to help users learn about cybersecurity in a fun and interactive way teaching safe online habits through conversation.
Features

Cybersecurity tips and guidance on a range of topics
Recognises keywords like:

Password Security
Phishing
Online Scams
Privacy
Malware
Safe Browsing
VPNs
Two-Factor Authentication

Detects how you're feeling and adjusts responses accordingly
Remembers your name and favourite topic throughout the conversation
Handles follow-up questions like "tell me more"
Clean GUI built with Windows Forms
Plays a voice greeting when the app starts

Technologies Used

C# and .NET
Windows Forms
Object-Oriented Programming

How to Run

Open the project in Visual Studio
Build the solution
Run the app
Enter your name when prompted
Start chatting using the text box or click one of the quick topic buttons

Example Questions

Tell me about phishing
How do I create a strong password?
What is two-factor authentication?
Tell me more about privacy
How does a VPN work?

Project Structure

ChatbotEngine.cs — all the chatbot logic and response handling
UserMemory — stores and recalls user information during the conversation
MainForm.cs — the graphical user interface
AudioPlayer.cs — handles the audio greeting on startup
